# AI Usage Report

## Project
SmartCheck – Smart Class Check-in & Learning Reflection App

## AI Tools Used
- ChatGPT

## How AI Was Used

AI was used as a coding assistant during the development of this project. It helped with several tasks including:

1. **Flutter Development**
   - Guidance on structuring Flutter project files
   - Creating UI pages such as login, register, and home page
   - Implementing navigation between screens

2. **Firebase Integration**
   - Connecting Flutter with Firebase Authentication
   - Storing and retrieving data using Cloud Firestore
   - Debugging Firestore queries and indexes

3. **Debugging and Troubleshooting**
   - Fixing errors related to Flutter and Firebase setup
   - Solving Git and GitHub deployment issues
   - Configuring Firebase Hosting deployment

4. **Documentation Assistance**
   - Helping generate the Product Requirement Document (PRD)
   - Structuring the README and project documentation

## Student Understanding

Although AI provided suggestions and example code, the student reviewed and modified the generated code to ensure it worked correctly with the application requirements.

The student understands:
- How Flutter widgets and navigation work
- How Firebase Authentication manages user login
- How Firestore stores and retrieves attendance data
- How to deploy a Flutter web application using Firebase Hosting

## Engineering Judgment

AI suggestions were not used blindly. The code was tested and adjusted to match the project requirements. The final implementation reflects the student's understanding of the system architecture and functionality.
